# WeChatBotAPI
微信使用的小机器人的api


更新点：

  星座运势 -- gc-1998
  喝水提醒 -- maotou



现有功能点：

用户：
  注册
  获取用户积分
  获取用户头像
  获取用户信息
  更新积分排名
  更新积分值
  更新所在城市
  更新签到记录

打卡：
  打卡签到
  打卡积分制度
  打卡生成图片

天气：
  预报天气：今日、一小时后天气

位置：
  给于地区：获取经纬度

模板生成：
  打卡模板生成图片
